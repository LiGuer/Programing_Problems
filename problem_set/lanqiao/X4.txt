在1至2019中，有多少个数的数位中包含数字9？
　　注意，有的数中的数位中包含多个9，这个数只算一次。例如，1999这个数包含数字9，在计算只是算一个数。
==================================================
544     对。
------------------------------------------------------------------------------------------
#include <iostream>
using namespace std;
typedef long long ll;

int main()
{
	int ans = 0;
	for (int i = 1; i <= 2019; i++) {
		int t = i;
		while (t) {
			if (t % 10 == 9) {
				ans++; break;
			}
			t /= 10;
		}
	}
	printf("%d\n", ans);
}
