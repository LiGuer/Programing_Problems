1：三角形面积
已知三角形三个顶点在直角坐标系下的坐标分别为：
(2.3, 2.5)
(6.4, 3.1)
(5.1, 7.2)
求该三角形的面积。
注意，要提交的是一个小数形式表示的浮点数。
要求精确到小数后3位，如不足3位，需要补零。
===============================================
8.795
-------------------------------------------------------------------------------------------
海伦-秦九韶公式。
-------------------------------------------------------------------------------------------
#include <iostream>
using namespace std;
struct position {
	double x, y;
};
int main()
{
	position a = { 2.3, 2.5 };
	position b = { 6.4, 3.1 };
	position c = { 5.1, 7.2 };
	double ab = sqrt((a.x - b.x) * (a.x - b.x) + (a.y - b.y) * (a.y - b.y));
	double cb = sqrt((c.x - b.x) * (c.x - b.x) + (c.y - b.y) * (c.y - b.y));
	double ac = sqrt((a.x - c.x) * (a.x - c.x) + (a.y - c.y) * (a.y - c.y));
	double p = (ab + cb + ac)/2;
	double S = sqrt(p * (p - ab) * (p - cb) * (p - ac));
	printf("%f\n", S);
}
